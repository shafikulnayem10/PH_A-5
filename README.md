# GitHub Issues Tracker 🐛

A web-based GitHub Issues Tracker application that allows users to sign in and manage their GitHub issues through a clean and simple interface.

## 🌐 Live Demo

🔗 [https://my-github-issues-tracker.netlify.app/](https://my-github-issues-tracker.netlify.app/)



## ✨ Features

- 🔐 User authentication (Sign In)
- 📋 View and manage GitHub issues
- 🎨 Clean and minimal UI
- 📱 Responsive design

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shafikulnayem10/PH_A-5.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd PH_A-5
   ```

3. **Open `index.html` in your browser:**
   ```bash
   open index.html
   ```
   Or simply double-click the `index.html` file.
